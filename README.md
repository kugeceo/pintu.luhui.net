## 鲁虺急速拼图
# pintu.luhui.net
速拼图-鲁虺急速拼图-照片拼图-在线拼图软件-模块拼图工具

在线模板拼图 | 提供多种拼接模式，支持调整边框、背景、图片角度等多种参数



## 功能优势

![速拼图绘制展示](http://pintu.luhui.net/img/strength/pintu.webp)


# 多种拼接模板

预置多种拼接的布局，轻松随心拼图
支持调整边距、间距等多项参数
可按需设置边框、间距、图片角度的大小等，满足多样化拼图需求

# 一键设置拼图的背景
支持自定义图片背景或纯色背景
